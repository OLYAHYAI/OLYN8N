Routine de Maintenance des VM Proxmox

Tags: #proxmox #virtualization #maintenance #automation

Référence Rapide

Tâche Fréquence Commande
Vérifier le stockage hôte Hebdomadaire df -h /
Vérifier l'utilisation du thin pool Hebdomadaire lvs pve/data
Examiner les snapshots Hebdomadaire qm listsnapshot <VMID>
Nettoyer les anciens snapshots Au besoin qm delsnapshot <VMID> <name>
Vérification des sauvegardes Mensuelle Check /var/lib/vz/dump/

Vérification de l'état du Stockage

Surveiller le Stockage Hôte

# Check root filesystem usage (keep below 90%)
df -h /

# Target: Keep usage below 85-90%
# Alert level: 90%+
# Critical: 95%+

Surveiller le Thin Pool LVM

# Check thin pool data usage
lvs pve/data

# Target: Keep data% below 80%
# Alert level: 80%+
# Critical: 90%+

# Full view with all volumes
lvs -a | grep data

Extension du Stockage (Si Nécessaire)

# Check available space in volume group
vgdisplay pve | grep -i free

# Extend root filesystem
lvextend -l +100%FREE /dev/pve/root
resize2fs /dev/pve/root

# Verify expansion
df -h /

Stratégie de Gestion des Snapshots

⚠️ Règle d'Or

Maintenir un maximum de 1 à 2 snapshots par VM à tout moment

Types de Snapshot

1. Snapshot de Référence (Golden Snapshot) (À Conserver)

  • Créé après une configuration majeure réussie
  • Représente un état stable et fonctionnel
  • Mise à jour rare (trimestrielle ou après des changements majeurs)
# Create golden snapshot
qm snapshot <VMID> golden-working-state

# Update golden (delete old, create new)
qm delsnapshot <VMID> golden-working-state
qm snapshot <VMID> golden-working-state

2. Snapshot Pré-Changement (Temporaire)

  • Créé avant des opérations risquées
  • Supprimer dans les 24 à 48 heures
  • Ne jamais en conserver plus d'un
# Before risky change
qm snapshot <VMID> pre-change-$(date +%Y%m%d)

# After successful change (DELETE IT!)
qm delsnapshot <VMID> pre-change-20251030

# If change failed (rollback then delete)
qm rollback <VMID> pre-change-20251030
qm delsnapshot <VMID> pre-change-20251030

Lister et Nettoyer les Snapshots

# List all snapshots for a VM
qm listsnapshot <VMID>

# Clean up old snapshots
qm delsnapshot <VMID> <snapshot-name>

# Example cleanup script
for snap in $(qm listsnapshot 100 | grep -v current | grep -v golden | awk '{print $2}'); do
    qm delsnapshot 100 $snap
done
