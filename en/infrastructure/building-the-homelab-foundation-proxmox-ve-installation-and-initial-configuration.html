2. üîå Ethernet Connection Between Router and Server

  • Manual IP configuration:

    • Netmask : 255.255.255.0
  • Goal : direct communication between the two machines

3. üì° Wi-Fi Check via Command Line

  • Commands used:
    • ip a ‚Üí list network interfaces
    • lspci | grep -i network ‚Üí detect network cards
    • iw dev ‚Üí Wi-Fi interfaces
    • lsusb ‚Üí USB devices (Wi-Fi dongle)

4. üåê Changing the Proxmox Server IP Address

  • Modified file : /etc/network/interfaces

  • Configuration example:

    auto vmbr0
iface vmbr0 inet static
    address 10.10.10.254
    netmask 255.255.255.0
    gateway 192.168.1.1
    bridge_ports enp3s0
    bridge_stp off
    bridge_fd 0

  • Network restart : systemctl restart networking

  • Methods :

  • Goal : create a VM using a customized ISO image
